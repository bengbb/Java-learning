# Java-learning
利用每天一点空余时间学习Java，在此记录学习过程.    
第一课    
学习java的安装,环境变量的配置,IDEA的下载和安装.    
第二课  
Hello World编写，IDEA基本设置以及插件安装.    
第三课  
注释相关内容学习，了解hello world中关键字，字面量.   
第四课  
学习变量，和c语言几乎一致.  
第五课  
学习了数据类型，标识符和键盘录入.  
第六课  
学习了算术运算符和类型转换.  
第七课  
学习了字符运算，字符串拼接，各类运算符以及if的基础.  
第八课  
学习了elseif语句以及相关练习.  
第九课  
学习了switch和for循环语句.
